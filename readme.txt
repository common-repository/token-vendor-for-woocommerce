== Description ==

Enables to generate token as product, which can be validated by the WC shop while it is valid and is not expired.


== Installation ==

<ol>
	<li> upload to "/wp-content/plugins/token-vendor-for-woocommerce", or directly search and install from wp repository on your admin page. ( "Plugins" -> "Add New" )</li>
	<li>Activate.</li>
</ol>

== Frequently Asked Questions ==

= How to use =
How-to page is prepared in admin page "WooCommerce" -> "NTVWC". Please regard.


= What this plugin for? =
Enable WooCommerce to sell token to be validate.
